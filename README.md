## Overview

This project presents a fully integrated 5-year financial model built in Excel for Trinity Inc. The model includes:

- Income Statement (5-Year Forecast)
- Cash Flow Statement (Indirect Method)
- Balance Sheet
- Assumptions & Drivers Sheet
- Dynamic Forecasting Structure
- Integrated Financial Logic

The purpose of this model is to evaluate financial performance, funding requirements, profitability trajectory, and investment viability.

---

## Model Structure

The financial model is built using standard financial modeling best practices:

1. Assumptions & Revenue Drivers
2. Revenue Forecasting
3. Cost Structure Modeling
4. EBITDA Calculation
5. Depreciation & Capital Expenditure Modeling
6. Working Capital Adjustments
7. Cash Flow Construction
8. Balance Sheet Balancing Logic

All three statements are fully linked and dynamically flow from assumptions to financial outputs.
